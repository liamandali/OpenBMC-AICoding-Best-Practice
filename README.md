# OpenBMC-AICoding-Best-Practice
## OpenBMC AI 辅助开发实践项目

本项目为基于OpenBMC生态的研发实践工程，基于人工智能辅助编程技术，围绕固件开发全流程开展体系化建设与工程化实践。项目聚焦智能体基础设施构建、基础能力沉淀与通用固件场景落地，形成可复用、可迁移、可推广的研发技术资产。

## 使用说明

本项目以技术实践沉淀与研发能力积累为核心目标，内容涵盖 AI 辅助开发环境配置、工具脚本、代码示例、设计文档及实践方案总结等。使用者可结合自身 OpenBMC 相关项目需求，对项目中的组件、方法及实践方案进行适配、集成与扩展，无需额外搭建完整工程环境即可复用相关技术资产。

## 适用范围

适用于 BMC 固件开发与维护、嵌入式底层软件开发、OpenBMC 二次开发，以及 AI 辅助编程技术在工业级嵌入式领域的落地实践与研究场景。
